# YouTube Dimmer 💡

A small Chromium extension that turns off the lights on YouTube.

With this extension, you can just enjoy the video (e.g., on your second monitor) while chilling in the dark.

## How it works
Click the extension icon -> everything goes black except the video player. Click again -> lights on. A genius admires simplicity.

## Installation (for personal use)
Since this isn't a public project and it never will be, it's not on the Chrome Web Store. Install it manually:

1. Download this repo (Code -> Download ZIP) or clone it via git.
2. Unzip if necessary.
3. Open a terminal in the project folder and run:

   ```bash
   npm install
   npm run build
5. In your browser, go to chrome://extensions/ or brave://extensions/.
6. Enable "Developer mode" (top right corner).
7. Click "Load unpacked" and select the build/chrome-mv3-prod folder inside the project.
   Done.
